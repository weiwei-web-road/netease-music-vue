# 网易云音乐
使用Vue全家桶设计和实现网易云音乐网站。项目中没有使用第三方的UI库。所有的组件仅仅针对本项目设计和实现。已经完成的功能有发现音乐、我的音乐、歌单详情页、音乐播放器、播放列表和歌词。

## Online Demo @ Daocloud
http://52.81.106.198:8080/
